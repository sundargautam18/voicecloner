Settings

Built-in voices work without any configuration. Cloning a voice from your own uploaded/recorded audio uses Kyutai's gated voice-cloning weights, which need a Hugging Face access token:

  1. Create an account at huggingface.co if needed.
  2. Accept the terms at huggingface.co/kyutai/pocket-tts.
  3. Create a read token at huggingface.co/settings/tokens and paste it below.

Alternatively, set the HF_TOKEN environment variable before starting the server, or run hf auth login in the activated virtual environment — either works instead of entering a token here.
